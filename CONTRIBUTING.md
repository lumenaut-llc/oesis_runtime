# Contributing

This repository is the Python reference runtime for OESIS. It is licensed under the **GNU Affero General Public License v3.0 or later** — see [`LICENSE`](LICENSE).

## Expectations

- Program contracts, schemas, and formal architecture live in **[oesis-program-specs](https://github.com/lumenaut-llc/oesis-program-specs)**. Align changes with that repository where applicable.
- Read the program’s contributor and publication guidance: [Contributor and review guide](https://github.com/lumenaut-llc/oesis-program-specs/blob/main/release/v1.0/contributor-and-review-guide.md).
- Before submitting changes, run `make oesis-check` when you can, or at least `make oesis-validate` plus the checks relevant to your edit.
